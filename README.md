## Installation

```bash
composer install
```

## Testing

The project is tested across multiple Symfony versions (7.x and 8.x) using GitHub Actions.

### Run PHPUnit tests

```bash
vendor/bin/phpunit
```

### Run Behat tests

```bash
vendor/bin/behat
```

### DemoBundle
